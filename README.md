# Powershell_Utilities
Repository containing an assortment of PowerShell scripts I've written.

movefiles-dateArrange.ps1
Takes a set of photos and videos nested in one folder or all in a single location and sorts them into new folders arranged by year and month. Also will rename the files to the date taken.

find-email-in-text.ps1
Searches through a text file and pulls out any email addresses found.
